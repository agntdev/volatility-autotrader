# Volatility AutoTrader

Telegram auto-trading bot for Deriv synthetic indices (V50/V75) using head-and-shoulders price-action, position-sizing and risk controls for small accounts.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
